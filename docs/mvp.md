# Job Match AI MVP

## Goal

Create a web app that helps a job seeker upload a CV, understand their strongest job-search angles, and generate high-quality LinkedIn job searches automatically.

## MVP flow

1. User uploads or pastes a CV.
2. The app extracts a professional profile.
3. User confirms target roles, seniority, location, work mode, and key skills.
4. The app generates LinkedIn search URLs.
5. The user opens LinkedIn with filters already prepared.
6. Later phases add job ingestion from open sources and a LinkedIn browser extension.

## Match labels

- Postulate ahora: strong fit, user should apply soon.
- Buena oportunidad: good fit, minor CV adjustments recommended.
- Posible: partial fit, useful if the user wants to stretch.
- Baja prioridad: weak fit or important missing requirements.

## First implementation slice

- Static home/dashboard.
- CV upload area.
- Preferences panel.
- Suggested LinkedIn searches.
- Product copy that explains the workflow without overpromising.
